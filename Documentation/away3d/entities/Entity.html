Packageaway3d.entities
Classpublic class Entity
InheritanceEntity ObjectContainer3D Object3D NamedAssetBase flash.events.EventDispatcher
Subclasses Camera3D, LightBase, Mesh, SegmentSet, SkyBox, Sprite3D

Класс Entity является абстрактным базовым классом для всех объектов сцены, которые можно считать "присутствующими" в сцене, в том смысле, что такой объект можно считать реальным объектом с позицией на сцене и размером (даже если он бесконечный или идеализированный), а не группой. Сущности добавляются в граф сцены и обслуживаются сборщиком сущностей EntityCollector.

getModelViewProjectionUnsafe()method 
public function getModelViewProjectionUnsafe():Matrix3D

Этот метод выполняет ту же функцию, что и геттер modelViewProjection. Но в отличие от него, в этом методе не происходит проверка вылета указателя за граница стека. Этот метод вызывается только внутри цикла рендеринга.

Returns
Matrix3D — MVP (Model-View-Projection)- матрица.

pushModelViewProjection()method 
public function pushModelViewProjection(camera:Camera3D):void

Обновляет MVP (Model-View-Projection)- матрицу, которая используется для проецирования объекта на проекционную плоскость и помещает ее в стек. Стек позволяет сохранять MVP-матрицы при визуализации объекта нетронутыми.

Parameters

camera:Camera3D — камера, матрица вида и матрица проекции которой будет использоваться при визуализации этого объекта.
