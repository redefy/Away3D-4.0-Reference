Окно статистики может использоваться для показа различной статистики в Away3D приложении или в любом другом приложении. Виджет, специально был разработан с целью показа всех необходимых данных таким образом, чтобы их легко можно было прочесть, но при этом чтобы сам виджет был небольшого размера.

Следующие данные показываются в виджете либо графически, либо с помощью текста, либо обоими вариантами.

  • Текущая частота кадров в FPS (белый график/бар)
  • Частота кадров SWF (Stage.frameRate)
  • Средний FPS (синий график / бар)
  • Минимальный/максимальный FPS (только в баре частоты кадров в минимизированном режиме)
  • Текущий размер занимаемой оперативной памяти (розовый график)
  • Максимальный размер занимаемой оперативной памяти
  • Количество полигонов в сцене
  • Количество полигонов визуализированных в последнем цикле рендеринга (желтый график)

Есть два режима отображения окна; стандартный и минимизированный, которые сменяют друг друга при клике по кнопке в верхнем правом углу. При создании виджета, можно сразу указать чтобы он запускался в минимизированном режиме установив соответствующий параметр в конструкторе виджета в true.

Все данные могут быть сброшены в любой момент, для этого необходимо кликнуть по нижней части виджета (там где находятся счетчики RAM и POLY. Средний FPS можно сбросить отдельно, кликнув по текстовому полю показывающему этот средний FPS. Кроме этого, частота кадров сцены может быть увеличена или уменьшена при клике по верхней или нижней части диаграммы. Чем ближе к центру диаграммы вы будете кликать мышкой, тем медленнее будет увеличиваться частота кадров сцены, чем дальше от центра диаграммы вы будете кликать мышкой тем быстрее будет увеличиваться частота кадров сцены. Все просто. Диаграмма показывается только в стандартном режиме отображения окна.

Средний FPS рассчитывается опираясь на значение одного их переданных параметров конструктора. Передав в параметр конструктора meanDataLength ненулевое значение, вы определите количество записей частоты кадров на основании которых будет расчитываться средний FPS. По умолчанию, этот параметр равен нулю, и в этом случае средний FPS будет рассчитываться на основе суммы всех записей частоты кадров начиная с последнего сброса статистики и по текущий момент.

AwayStats()Constructor
public function AwayStats(view3d:View3D = null, minimized:Boolean = false, transparent:Boolean = false, meanDataLength:uint = 0, enableClickToReset:Boolean = true, enableModifyFrameRate:Boolean = true)

Создает новый виджет статистики Away3D. Виджет может быть добавлен на сцену и позиционироваться также как и любой другой экранный объект. После того как виджет будет добавлен на сцену вы можете перетащить его в удобное для вас место на сцене.

Если вы передаете ссылку на экземпляр класса View3D, то виджет будет показывать общее количество полигонов находящихся в сцене и количество полигонов визуализированных за время последнего вызова метода render(). Вьюпорты в виджете также могут быть зарегистрированны при помощи метода registerView(). Не передавайте ссылку на вьюпорт в параметр view конструктора если хотите отключить эту возможность полностью.

Parameters
view3d:View3D (default = null) — Ссылка на ваш Away3D вьюпорт. Это необходимо если вы хотите видеть количество полигонов находящихся в данный момент в сцене.
 
minimized:Boolean (default = false) — Определяет будет ли виджет запускаться в минимизированном режиме. По умолчанию, виджет запускается в стандартном режиме.
 
transparent:Boolean (default = false) — Определяет будет ли у виджета рисоваться фон.
 
meanDataLength:uint (default = 0) — Количество кадров на основе которых расчитывается средняя частота кадров. Значение по умолчанию 0, означает что учитываются все кадры начиная с момента последнего сброса счетчиков.
 
enableClickToReset:Boolean (default = true) — Активирует возможность сброса всех счетчиков при клике по нижнему бару виджета. Когда эта возможность активирована вы также можете сбрасывать значение средней частоты кадров просто кликая по текстовому полю показывающее это значение.
 
enableModifyFrameRate:Boolean (default = true) — Активирует возможность изменения частоты кадров SWF при клике по верхней или нижней части диаграммы.

registerView()method
public function registerView(view3d:View3D):void

Добавляет вьюпорт в список тех вьюпортов, которые учитываются при расчете общего количества полигонов отрисовывающихся на экране. Используйте этот метод в случаях когда ваш виджет не создается в том же самом месте где и ваш вьюпорт или когда вы используете несколько вьюпортов, или когда вьюпорты создаются и уничтожаются динамически, во время работы приложения.

Parameters

view3d:View3D — вьюпорт.

public function reset():void

unregisterView()method 
public function unregisterView(view3d:View3D):void

Удаляет вьюпорт из списка тех вьюпортов, которые учитываются при расчете общего количества полигонов отрисовывающихся на экране. Если этот вьюпорт является единственным вьюпортом известным виджету, то при вызове этого метода, вектор _views станет пустым и вся статистика по количеству полигонов рисующихся на экране будет отключена.
