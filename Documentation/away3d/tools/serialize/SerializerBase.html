Packageaway3d.tools.serialize
Classpublic class SerializerBase
InheritanceSerializerBase Object
Subclasses TraceSerializer

Класс SerializerBase это абстрактный базовый класс для всех классов-сериализаторов. Он предоставляет интерфейс для осуществления сериализации основных типов данных движка. Десериализация данных не поддерживается.
