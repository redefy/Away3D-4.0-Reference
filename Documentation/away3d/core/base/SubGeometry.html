Packageaway3d.core.base
Classpublic class SubGeometry
InheritanceSubGeometry Object
Subclasses SkinnedSubGeometry

Класс SubGeometry содержит все необходимые геометрические данные, требующиеся для создания полигонального меша. Этот класс принадлежит экземпляру класса Geometry, и передается экземпляру класса SubMesh, который является узлом графа сцены. Несколько экземпляров класса SubGeometry могут быть сгруппированы так, что будут рендериться с различными материалами, но по прежнему представлять из себя один цельный объект.
