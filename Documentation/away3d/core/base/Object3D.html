Object3D является базовым классом для любого 3D объекта к которому можно применять какие-то (локальные) трансформации.

Стандартные трансформации:

  • - Стандартная трансформация объекта расчитывается по такой схеме [трансформация родителя] * (перемещение+точка привязки) * (вращение) * (-точка привязки) * (масштаб) * [трансформация ребенка].
  • - Перемножение матриц происходит слева-направо.
  • - Трансформация же к объекту применяется справа-налево! (Масштаб) применяется к объекту до (-точки привязки) она в свою очередь до (вращения), а вращение применяется к объекту до (перемещение+точка привязки)
  • - Без точки привязки трансформация расчитывается по такой схеме [трансформация родителя] * перемещение * вращение * масштаб * [трансформация ребенка]. (Масштаб) применяется к объекту до (вращения), а вращение применяется к объекту до (перемещения)
  • - Код реализующий алгоритм выше находится в методе updateTransform этого объекта и в методе ObjectContainer3D.updateSceneTransform().
  • - При использовании метода prepend() умножение матриц выполняется с правой стороны: transform = transform * rhs;
  • - При использовании метода append() умножение матриц выполняется с левой стороны: transform = lhs * transform;

Для изменения масштаба объекта:

  • - используйте свойства scaleX/Y/Z или метод scale(delta).

Для изменения точки привязки:

  • - используйте свойство pivotPoint или метод movePivot().

Для изменения вращения объекта:

  • - используйте свойства rotationX/Y/Z (углы указываются в градусах), свойcтво eulers [необходимо указать сразу 3 угла] (углы указываются в радианах), или вызовите метод rotateTo().
  • - используйте методы pitch()/yaw()/roll()/rotate() для поворота объекта вокруг локальных осей. Свойства rotationX/Y/Z будут обновлятся автоматически при вызове этих методов.

Для перемещения объекта:

  • - используйте свойства x/y/z/position или вызовите метод moveTo().
  • - используйте метод translate(), который модифицирует свойства объекта x/y/z основываяcь на значениях переданного вектора.
  • - используйте методы moveForward()/moveBackward()/moveLeft()/moveRight()/moveUp()/moveDown()/translateLocal() для перемещения объекта вдоль локальных осей. Свойства x/y/z будут обновлятся автоматически при вызове этих методов.

Parameters
