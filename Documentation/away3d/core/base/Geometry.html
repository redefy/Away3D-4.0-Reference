Packageaway3d.core.base
Classpublic class Geometry
InheritanceGeometry NamedAssetBase flash.events.EventDispatcher
Implements IAsset
Subclasses PrimitiveBase

Класс Geometry содержит коллекцию объектов SubGeometry, в каждом из которых находятся фактические геометрические данные, такие как вершины, нормали, текстурные координаты, и т.д. Он также содержит ссылку на класс анимации, который определяет как должна анимироваться геометрия. Каждым объектом Geometry владеет объект Mesh, который является узлом графа сцены. В свою очередь, каждым объектом SubGeometry владеет объект SubMesh, на который можно назначить материал с которым он будет визуализирован.
