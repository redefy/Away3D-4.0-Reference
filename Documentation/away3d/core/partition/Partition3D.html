Classpublic class Partition3D
InheritancePartition3D Object
Subclasses Octree, QuadTree

Partition3D является ядром системы разбиения пространства. Система разбиения пространства разбивает одну отдельную 3D-сцену на некоторое количество непересекающихся подпространств, формируя их в дерево (структура данных). Это нужно для реализации отсечения по пирамиде видимости, определения видимости объектов и обнаружении столкновений.
