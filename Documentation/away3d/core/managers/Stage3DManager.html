Packageaway3d.core.managers
Classpublic class Stage3DManager
InheritanceStage3DManager Object

Класс Stage3DManager представляет из себя менеджер, который управляет объектами Stage3D. Вам не следует обращаться к объектам Stage3D напрямую, вместо этого вы должны производить все манипуляции с объектами Stage3D через экземпляры классов Stage3DProxy.
