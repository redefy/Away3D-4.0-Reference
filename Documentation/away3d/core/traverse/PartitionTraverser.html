entryPointproperty
entryPoint:Vector3D  [read-only]

Отправная точка для обхода графа сцены, то есть это та точка которая будет использоваться для позиционно-зависимого обхода графа сцены. Например: BSP определение видимости или обнаружение столкновений. В EntityCollector, эта точка будет содержать позицию камеры на сцене.

enterNode()method 
public function enterNode(node:NodeBase):Boolean

Возвращает true, если указанный узел хотя бы частично находится в фрустуме камеры. При этом все дети этого узла и сам узел будут добавлены в текущий конвейер сцены.

Parameters

node:NodeBase — узел для проверки.

Returns
Boolean — true, если указанный узел хотя бы частично находится в фрустуме камеры.
leaveNode()method 
public function leaveNode(node:NodeBase):void

Вызывается когда узел покидает конвейер. Этот метод также вызывается когда enterNode() возвращает false.

Parameters
