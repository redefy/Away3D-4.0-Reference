enterNode()method 
override public function enterNode(node:NodeBase):Boolean

Возвращает true, если указанный узел хотя бы частично находится в фрустуме камеры. При этом все дети этого узла и сам узел будут добавлены в текущий конвейер сцены.

Parameters

node:NodeBase — узел для проверки.

Returns
Boolean — true, если указанный узел хотя бы частично находится в фрустуме камеры.
