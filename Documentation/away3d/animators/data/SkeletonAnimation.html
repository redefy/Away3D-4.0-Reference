Packageaway3d.animators.data
Classpublic class SkeletonAnimation
InheritanceSkeletonAnimation AnimationBase Object

SkeletonAnimation defines an animation type that blends different skeletal poses into a final pose and transforms the geometries' vertices along the skeleton. Each vertex is bound to a number of skeleton joints with a number of weights, which define how the skeleton influences the vertex position.

usesCPUproperty 
usesCPU:Boolean  [read-only] [override]

Indicates whether or not the vertex transformation happens on CPU or GPU. Returns true if running on the CPU. This can happen when the animation cannot run on GPU, due to too many joints or too many joints per vertex.
