Packageaway3d.containers
Classpublic class Scene3D
InheritanceScene3D flash.events.EventDispatcher

Класс Scene3D представляет из себя независимую 3D сцену в которую могут быть добавлены 3D-объекты. В одном SWF-файле могут быть созданы несколько экземпляров класса Scene3D. Управление сценой происходит через граф сцены, добавление узлов к которому осуществляется с помощью методов addChild и removeChild сцены. Внутренне, объект Scene3D также управляет деревьями, которые могут использовать различные алгоритмы разделения пространства. В сцене всегда существует по крайней мере одно дерево.
