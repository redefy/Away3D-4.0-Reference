Создает новый экземпляр класса CubeGeometry.

Parameters
width:Number (default = 100) — ширина куба. Значение по умолчанию: 100.
 
height:Number (default = 100) — высота куба. Значение по умолчанию: 100.
 
depth:Number (default = 100) — длина куба. Значение по умолчанию: 100.
 
segmentsW:uint (default = 1) — количество сегментов из которых состоит куб по оси X. Значение по умолчанию: 1.
 
segmentsH:uint (default = 1) — количество сегментов из которых состоит куб по оси Y. Значение по умолчанию: 1.
 
segmentsD:uint (default = 1) — количество сегментов из которых состоит куб по оси Z. Значение по умолчанию: 1.
 
tile6:Boolean (default = true) — способ расчета UV-координат. Если true, текстура будет разделена на сетку 2x3, каждая часть которой будет наложена на какую-то одну сторону куба. Если false, то на каждую из сторон будет наложено все изображение целиком. Значение по умолчанию: true.
