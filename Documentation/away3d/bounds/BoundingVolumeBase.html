Classpublic class BoundingVolumeBase
InheritanceBoundingVolumeBase Object
Subclasses AxisAlignedBoundingBox, BoundingSphere, NullBounds

Абстрактный базовый класс для всех Bounding Volume(ограничивающий объем) классов. Каждому объекту сцены назначается экземпляр класса BoundingVolumeBase. Например, для мешей назначается AxisAlignedBoundingBox, для источников света BoundingSphere. При этом размер экземпляра класса BoundingVolumeBase выбирается таким образом, чтобы объект сцены полностью оказался вписан в этот объем. В дальнейшем объект BoundingVolumeBase закрепленный за объектом участвует в некоторых важных расчетах, например, отсечении геометрии по пирамиде видимости камеры, в расчете пересечения с лучом и т.д. Не следует создавать экземпляр этого класса вручную.
