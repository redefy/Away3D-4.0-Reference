Packageaway3d.bounds
Classpublic class BoundingSphere
InheritanceBoundingSphere BoundingVolumeBase Object

BoundingSphere представляет из себя ограничивающую сферу, которая описывается точкой-центром сферы и ее радиусом. Этот ограничивающий объем может быть полезен для точечных источников света.
