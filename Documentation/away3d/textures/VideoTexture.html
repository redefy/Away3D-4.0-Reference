public function VideoTexture(source:String, materialWidth:uint = 256, materialHeight:uint = 256, loop:Boolean = true, autoPlay:Boolean = false, player:IVideoPlayer = null)

Создает новый экземпляр класса VideoTexture.

Parameters
source:String — путь к видеофайлу.
 
materialWidth:uint (default = 256) — ширина видео.
 
materialHeight:uint (default = 256) — высота видео.
 
loop:Boolean (default = true) — определяет, будет ли видео повторяться снова и снова или же будет воспроизводится только один раз.
 
autoPlay:Boolean (default = false) — определяет, будет ли видео воспроизводится автоматически сразу после инициализации.
 
player:IVideoPlayer (default = null) — ссылка на видеоплеер.

Перерисовывает текстуру. Если свойство autoUpdate установлено в false и эта функция не вызывается вручную, то текстура не будет обновляться.
