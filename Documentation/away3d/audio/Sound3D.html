Packageaway3d.audio
Classpublic class Sound3D
InheritanceSound3D ObjectContainer3D Object3D NamedAssetBase flash.events.EventDispatcher

Объект-источник звука. Именно в той позиции где находится этот объект, громкость звука будет максимальной. Текущая громкость и панорамирование звука расcчитывается на основе позиции этого объекта относительно объекта-слушателя.

Работа с объектом Sound3D почти ничем не отличается от работы с примитивами, источниками света и камерами. Объект Sound3D точно также может быть добавлен к сцене и перемещен на желаемую позицию. Это главный объект в API 3D-звука в Away3D 4.0, и программист для управления 3D-звуком по сути будет взаимодействовать только с ним.

Реальная трансформация звука (громкость/панорамирование) осуществляется драйвером звука, который может быть передан в параметр driver конструктора этого класса. По умолчанию, используется драйвер SimplePanVolumeDriver.

volumeproperty 
volume:Number

Общая громкость (Master Volume) звука, которая применяется после всех 3D-модификаций громкости звука. Изменяйте это свойство для уменьшения или увеличения общей громкости звука независимо от нахождения источника звука. Это значение также можно изменять через драйвер звука, который использует этот объект Sound3D.

Constructor Detail
Sound3D()Constructor
public function Sound3D(sound:Sound, reference:ObjectContainer3D, driver:ISound3DDriver = null, volume:Number = 1, scale:Number = 1000)

Создает экземпляр класса Sound3D, который представляет из себя источник звука, через который воспроизводится нативный объект Sound.

Parameters
sound:Sound — объект Sound, который воспроизводится через этот источник звука. Для достижения реалистичных результатов, используйте моно (одноканальный, не стерео) звуковой поток.
 
reference:ObjectContainer3D — объект-слушатель. Как правило это камера.
 
driver:ISound3DDriver (default = null) — драйвер звука, который используется для управления звуком. По умолчанию, используется драйвер SimplePanVolumeDriver.
 
volume:Number (default = 1) — общая громкость звука.
 
scale:Number (default = 1000) — произвольное значение, на которое делятся все параметры связанные с дистанцией.

pause()method
public function pause():void

play()method 
public function play():void

Начинает воспроизведение звука (или возобновляет после паузы).

stop()method 
public function stop():void

Останавливает и перематывает звуковой файл на начало. Для возобновления воспроизведения звука (с начала) используйте метод play(). Чтобы временно остановить проигрывание звука, а потом продолжить воспроизведение с этой же точки, используйте метод pause().

pause()
togglePlayPause()method 
public function togglePlayPause():void

Ставит воспроизведение звука на паузу или продолжает воспроизведение звука после паузы. При вызове этого метода когда звук стоит на паузе, воспроизведение звука будет продолжено. При вызове этого метода когда звук проигрывается, воспроизведение звука ставится на паузу.

update()method 
public function update():void

Обновляет значения вектора, который содержит в себе позицию источника звука относительно слушателя. На основе значений этого вектора предписывает драйверу звука пересчитать параметры панорамирования и громкости звука.
