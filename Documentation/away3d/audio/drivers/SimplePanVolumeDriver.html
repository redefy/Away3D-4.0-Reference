Classpublic class SimplePanVolumeDriver
InheritanceSimplePanVolumeDriver AbstractSound3DDriver Object
Implements ISound3DDriver

Простой звуковой драйвер, который рассчитывает громкость и панорамирование звука относительно текущего положения и поворота слушателя относительно источника звука. Недостаток этого драйвера в том, что он учитывает азимут и расстояние, но не учитывает высоту.

scaleproperty
scale:Number[override]

Произвольное значение, на которое делятся все параметры связанные с дистанцией. Значение 1000, используемое по умолчанию, обычно подходит для сцен, масштаб которых примерно соответствует стандартному масштабу Away3D, т.е для тех сцен которые хорошо смотрятся с позиции по умолчанию камеры.

volumeproperty 
volume:Number[override]

Общая громкость (Master Volume) звука, которая применяется после всех 3D-модификаций громкости звука. Изменяйте это свойство для уменьшения или увеличения общей громкости звука независимо от нахождения источника звука.

pause()method
public function pause():void

Временно приостанавливает воспроизведение звука. Для возобновления воспроизведения звука используйте метод play().

play()method 
public function play():void

Начинает воспроизведение звука (или возобновляет после паузы). Это не тоже самое что и вызов метода play() объекта flash.media.Sound, который используется в качестве источника звука драйвера.

stop()method 
public function stop():void

Останавливает воспроизведение звука. При последующем вызове метода play(), воспроизведение звука начнется с начала звукового файла.

updateReferenceVector()method 
override public function updateReferenceVector(v:Vector3D):void

Изменяет позицию источника звука относительно слушателя. Позиция звука указывается с помощью вектора. Этот метод вызывается объектом Sound3D в случае когда позиция источника звука меняется, и как правило не должен вызываться больше не из каких других мест.
